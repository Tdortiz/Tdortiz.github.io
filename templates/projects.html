Door To More
Off The Grid : Blackbaud. Summer 2016 - finalist
Door To More was created in 24 hours to benefit advocacy organizations that would employ individuals to canvass door to door or on the street to raise awareness, solicit donations, and add to in-house files. The purpose of the app is to create a streamlined interface where canvassers can quickly create new constituents and document the interaction in a way that advocacy organizations can aggregate or target these constituents in the future.

Research File Anonymizer
Created by Thomas Ortiz. Spring 2016
This java program was developed to quickly anonymize student files by removing identifying information such as email, names, and @author tags and replacing them with encrypted ids generated by the research director. As the program parses through student files it creates new anonymized files and saves the new files into a directory called Anon_Submissions while preserving the old copies of the files.
